## 安裝

```shell
$ npm install
```

## 使用方法

先[下載](#下載測試檔案)該有的檔案，然後執行下列程式碼。

```shell
# only many time attack
$ npm run many-time-attack
# hashing video
$ npm run hash-video
# only padding oracle attack
$ npm run padding-oracle-attack
# all test
$ npm run test
```
