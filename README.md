- Many Time Attack - 2020-10-22
- Hashing Videos - 2020-12-07

## 安裝

```shell
$ npm install
```

## 使用方法

```shell
# only many time attack
$ npm run mta
# hashing video
$ npm run hash
# all test
$ npm run test
```
